4 steps — that's it

After a one-time setup, it's a one-minute job every day.

Install

Download the setup and click Next-Next-Install. A desktop shortcut appears.

Add ffmpeg

Copy ffmpeg.exe + ffprobe.exe into the ReelCutter install folder (link below). One time only.

Pick a video & Start

Choose a video or paste a link, type a title, press Start — clips + captions are ready.

Upload / Share

Schedule everything to YouTube with ⬆, or send clips to your phone via QR. Done!

Requirements

Simple — only one thing is mandatory.

  • Windows 10/11 (64-bit) REQUIRED
  • ffmpeg — download here, then copy ffmpeg.exe + ffprobe.exe into the install folder:
    C:\Users\<name>\AppData\Local\Programs\ReelCutter REQUIRED
  • yt-dlp — put yt-dlp.exe in the same folder — only needed for the "cut from a link" feature OPTIONAL
  • Google setup — only for YouTube auto-upload (instructions inside the app) OPTIONAL

FAQ

The questions everyone asks.

Windows says "protected your PC"?

That happens with every new independent app (it isn't code-signed yet). Click More info → Run anyway. The app is 100% safe — it collects nothing; everything stays on your PC.

Is it really free?

Yes, completely free. No account, no subscription, and no watermark forced on your output (only your own logo, if you add one 😄).

How many YouTube uploads per day?

Google's default quota allows ~6 uploads/day. In schedule mode the app uploads clips as private and YouTube publishes one per day automatically — so the quota is rarely a problem.

Setup vs Portable — which one?

Setup is best for normal use (shortcuts + uninstaller). Portable runs without installing — even from a USB stick.

Is my data/account safe?

Yes. The app uploads/collects nothing on its own. Your YouTube login token is stored only on your PC (~/.reelcutter_yt_token.json). The source code is sealed inside the app — nobody can modify it.
