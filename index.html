FAQ

Jo sawal sabse zyada aate hain.

Windows "protected your PC" warning dikha raha hai?

Ye har naye independent software ke saath hota hai (app abhi code-signed nahi hai). More info → Run anyway dabao. App 100% safe hai — koi data collect nahi hota, sab kuch tumhare PC pe hi rehta hai.

Kya ye sach me free hai?

Haan, bilkul free. Koi account, koi subscription, koi watermark-on-output nahi (sirf tumhara apna logo, agar lagao to 😄).

YouTube pe roz kitne upload ho sakte hain?

Google ka default quota ~6 upload/din ka hai. App schedule mode me clips ko private upload karke roz ek publish karta hai — quota ki tension nahi.

Setup vs Portable — kaunsa loon?

Setup normal use ke liye best (shortcuts + uninstaller). Portable bina install ke chalta hai — pen-drive me bhi.

Mera data/account safe hai?

Haan. App kuch bhi upload/collect nahi karta. YouTube login ka token sirf tumhare PC pe (~/.reelcutter_yt_token.json) save hota hai. Source code app ke andar sealed hai — koi modify nahi kar sakta.
