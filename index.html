Questions

FAQ

The ones everyone asks.

Windows says "protected your PC"?

That happens with every new independent app (it isn't code-signed yet). Click More info → Run anyway. The app is 100% safe — it collects nothing; everything stays on your PC.

Is it really free?

Yes, completely free. No account, no subscription, and no forced watermark on your output — only your own logo, if you add one.

How many YouTube uploads per day?

Google's default quota allows ~6 uploads/day. In schedule mode the app uploads clips as private and YouTube publishes one per day automatically — so the quota is rarely a problem.

Setup vs Portable — which one?

Setup is best for almost everyone — ffmpeg is bundled, shortcuts + uninstaller included, zero configuration. Portable is a single small .exe that runs without installing (even from a USB stick), but you must place ffmpeg.exe + ffprobe.exe next to it yourself.

Why is the Setup ~113 MB?

Because the full ffmpeg video engine is packed inside. That's exactly why there's no "step 2" — you install and it just works.

Is my data / account safe?

Yes. Your videos, captions and YouTube login token never leave your PC. The only thing the app sends is a tiny anonymous launch ping (app version number only — no personal data, no file info) so we know how many people use each version. The source code is sealed inside the app — nobody can modify it.
